Google Analytics Setup

Enable servicesâ€‹

  1. Visit Google Api Console
  2. Search product 'Google Analytics Api'
  3. Click enable

Getting credentials JSONâ€‹

  1. Go to Google Analytics page.

  2. Login using Google Account.

  3. If you dont have an Analytics account before, fill in the form like image below.

    • Fill in account name. On Account Data Sharing Settings, check options that you want.
    • Fill in poperty name, reporting time zone and currency.
    • Click Show advanced options.
    • Fill in website URL with your site URL. Don't forget to check Create a Universal Analytics property only.
    • Check anything that your business is about.
    • After that, click the Create button and modal will appear then press I Accept button.

  4. Go to Google API and select a project that you want from the header.

  5. Go to Credential and create a new credential. Click Create Credentials and select Service account.

  1. Fill in the form with service account name and account ID that you like. After that, click Create button and click Done.

  1. To obtain service account credential JSON, press edit on the new created service account.

  2. Select KEYS menu from the tab. Click Add Key and select Create new key. After that, select JSON and click Create button.

  3. After you create the key, it'll automatically download the key.

  4. Place your .json key to storage directory like below.

Granting permissions to your Analytics propertyâ€‹

  1. Go to Google Analytics page.
  2. Select Admin menu from sidebar. Select View User Management.
  3. New window will appear, after that click Add users.
  4. Open the credential that we get before, search for client_email. Copy the email.
  5. Paste it in the Email addresses field and select permission that you want. You must check the Read & Analyze permissions. After that, press Done button.

Getting the ID that you need.â€‹

View IDâ€‹

  1. Select Admin menu from sidebar and select View Settings.

  2. There you have it.

Property ID / Tracking IDâ€‹

  1. Select Admin menu from sidebar and there you have it.

Account IDâ€‹

  1. Select Admin menu from sidebar and select Account Settings.

  2. There you have it.

Web Property IDâ€‹

  1. Open Google Analytics page.

  2. Look at the page URL. The web property ID start with w. Usually it has 9 character. For example: https://analytics.google.com/analytics/web/?authuser=1#/report-home/a000000000w299999997p000000000

    The web property ID for that account is 299999997.
